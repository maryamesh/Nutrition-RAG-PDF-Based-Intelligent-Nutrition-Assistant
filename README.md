# Nutrition-RAG-PDF-Based-Intelligent-Nutrition-Assistant
A Retrieval-Augmented Generation (RAG) system for nutrition PDFs. Uses Voyage AI embeddings (voyage-3), Pinecone as a vector database for semantic search, and the LLM meta-llama/llama-3.1-8b-instruct via OpenRouter for grounded answers. Includes PDF ingestion, chunking, embedding, retrieval, and an interactive Streamlit GUI chat interface.
